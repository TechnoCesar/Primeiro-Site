## 📝 Sobre o Projeto
Este é o meu primeiro repositório onde estou guardando meus estudos de HTML. O plano é atualizar este site com base no que vou aprendendo na faculdade de **Análise e Desenvolvimento de Sistemas**, colocar meus conhecimentos em teste e me divertir um pouco enquanto programo.

## 📂 Estrutura do Projeto
O projeto está organizado em pastas para facilitar a manutenção:
* **`/` (raiz)**: Contém o arquivo principal `index.html`.
* **`/pages.html`**: Contém as páginas secundárias, como a `sobremim.html`.
* **`/imgs`**: Reservada para todas as imagens.
